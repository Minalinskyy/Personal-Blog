Lost Object Web Site

This was my first project of web site programming. Based of PHP5 without framework and MySQL. It had a login system and it was build for the students of a school to declare lost private objects and to look for if someone else found it and posted it on this website. There was also an administrator who can manage the found and declared objects.

Everyone can have a try of it because I put it in a server. You can go to this site by the link on the top of page (Complete Project)

Web Crawler

I did this project by myself only because my school's schedule pushing stream is always so unstable. So I tried to use Python to write a script to download the information of my schedule and save them into a .ics file as the source of my GoogleCalendar. What a clever idea :)

In this project I studied a lot of Python programming and third-party libraries. I used Selenium to download the data and BeautifulSoup4 to analyse them. And also to make this script automatically run in the server, I got the usage of crontab in linux system.

A real-time face recognition with Deep-Learning

This was a project finally realised on Raspberry Pi. I built a model of CNN (Convolutional Neural Network) and trained it with the photos of my group members. Finally I used this model with a script written by myself on Raspberry Pi with an Pi-camera. The accuracy of the face recognition was over 90%.
